# Critérios de qualidade
Para a classe Student foi criado o método calculateAverage() onde é calculado a média de um aluno e é armazenado a sua média no seus próprio objeto.
Seguindo os critérios de encapsulamento foi retirado os métodos set's da classe Date tornando a classe imutável, pois não é conveniente alterar os atributos desta classe. Para classes Employee.java e Dependent.java foi criada uma classe abstrata chamada Person.java pois as duas classes tinham em comum atributo nome.
